### Help
    
```Bash
$ python                # open terminal 
>>> help()                # open interactive help
>>> quit()                # exit interactive help

>>> help(len)             # help(CommandName)

>>> help(math)            # help(module)
>>> help(math.factorial)  # help(function)

>>> a = 5
>>> help(a)               # help(object)
```
